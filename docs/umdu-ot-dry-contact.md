# Режимы работы реле сухого контакта

Реле сухого контакта в umdu ot — это дополнительный выход для управления внешней цепью. Оно удобно, когда нужно не только работать по `OpenTherm`, но и отдельно переключать насос, каскадный вход или другую внешнюю автоматику.

<img src="/images/umdu-ot-dry-contact.jpg" alt="Контакты реле сухого контакта umdu ot" class="doc-compact-image">

На реле есть три контакта:

- `COM` — общий контакт;
- `NO` — нормально разомкнутый контакт, замыкается с `COM`, когда реле включено;
- `NC` — нормально замкнутый контакт, замкнут с `COM`, когда реле выключено.

::: warning Важно
Если нужно коммутировать мощную или сетевую нагрузку, безопаснее использовать внешний контактор или промежуточное реле, а реле модуля оставить только как управляющий сухой контакт.
:::

## Режим 1 - Ручное управление

Подходит для сценариев:

- ручное включение внешней автоматики;
- автоматизация в Home Assistant;
- тестовая проверка подключения;
- редкие переключения по команде пользователя.

## Режим 2 - Дополнительный насос

Используйте этот режим, если нужно автоматически включать и отключать внешний насос вместе с отоплением.

Для переключения сухого контакта в этот режим в `Настройках` найдите `Настройки дополнительного насоса`. Выберите чекбокс `Использовать доп. насос` и нажмите `Сохранить`.

Для этого режима также доступны параметры:

- `Время постциркуляции` в минутах;
- `Интервал защиты от блокировки` в днях;
- `Время работы насоса` в минутах.

## Режим 3 - Каскадный выход

Используйте этот режим, если нужно передавать состояние модуля во внешнюю автоматику или котел, поддерживающий каскадный вход.

Для переключения сухого контакта в этот режим в `Настройках` найдите `Настройки каскадного управления`. Выберите чекбокс `Включить выход` и нажмите `Сохранить`.

Для этого режима также доступен параметр:

- `Пороговое время изменения состояния` в секундах.

Можно выбрать события, на которые будет реагировать реле:

- если состояние `fault` (`ошибки`) активно;
- если соединение по `OpenTherm` потеряно;
- если отопление включено.

## Важное ограничение режимов

Реле физически одно, поэтому автоматические режимы взаимоисключающие:

- нельзя одновременно использовать `Дополнительный насос` и `Каскадный выход`;
- если активен любой автоматический режим, ручное управление реле недоступно;
- если хотите вернуться к ручному переключению, сначала выключите активный автоматический режим.
